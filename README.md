# Каталог образовательных программ для участников СВО

## Описание проекта

Интерактивная веб-страница с каталогом образовательных программ, ориентированных на профессиональную ресоциализацию участников СВО через обучение по востребованным профессиям.

## Особенности

- Интерактивный каталог по 10 макрогруппам профессий
- Фильтры по уровню образования, форме обучения и региону
- Информация о льготах для участников СВО
- Прямые ссылки на ФГОС и профессиональные стандарты
- Адаптивный дизайн для мобильных устройств
- Поиск по профессиям и специальностям

## Структура

- Введение с информацией о дефиците кадров и льготах для участников СВО
- Методология отбора программ
- Интерактивный каталог с 10 макрогруппами
- Фильтры и персонализация
- Нормативно-правовая база

## Технологии

- HTML5
- CSS3 (с использованием Flexbox и Grid)
- JavaScript (ES6+)
- Адаптивный дизайн

## Установка и запуск

1. Клонируйте репозиторий:
```bash
git clone <ваш_репозиторий>
```

2. Откройте файл `index_svo.html` в браузере

## Структура проекта

```
├── index_svo.html      # Основная HTML страница каталога для СВО
```

## Каталог включает 10 макрогрупп профессий:

1. IT и цифровые технологии
2. Здравоохранение и биотехнологии
3. Энергетика и энергомашиностроение
4. Транспорт и логистика
5. Строительство и архитектура
6. Промышленность и материаловедение
7. Сельское хозяйство
8. Экономика и управление
9. Право и юриспруденция
10. Образование и педагогика

## Льготы для участников СВО

- Отдельная квота при приёме (ст. 68 ФЗ-273)
- Бюджетные места в государственных учреждениях (ст. 71 ФЗ-273)
- Внеочередное зачисление (ст. 68 и 71 ФЗ-273)
- Возможность увеличения срока обучения для инвалидов и лиц с ОВЗ (макс. +1 год)

## Нормативно-правовая база

- ФЗ-273 «Об образовании в Российской Федерации»
- Приказы Минобрнауки и Минпросвещения
- Общероссийский классификатор специальностей и профессий (ОКСО)
- Трудовой кодекс РФ и требования к применению профессиональных стандартов
