# Установка и запуск каталога образовательных программ для участников СВО

## Требования

- Node.js (для запуска локального сервера)
- Современный веб-браузер

## Установка

1. Установите Node.js с официального сайта: https://nodejs.org/
2. Скачайте или склонируйте репозиторий с проектом

## Запуск

### Вариант 1: Использование npm (рекомендуется)

```bash
# Установите зависимости
npm install

# Запустите локальный сервер
npm start
```

После запуска проект будет доступен по адресу: http://localhost:3000

### Вариант 2: Прямое открытие в браузере

Откройте файл `index_svo.html` в веб-браузере напрямую.

## Использование

1. Перейдите на главную страницу каталога
2. Используйте фильтры для поиска подходящих образовательных программ
3. Раскройте интересующие категории для просмотра деталей программ
4. Кликните по ссылкам на профессиональные стандарты для получения дополнительной информации

## Структура проекта

```
├── index_svo.html      # Основная HTML страница каталога для СВО
├── styles_svo.css      # Стили каталога для СВО
├── script_svo.js       # Логика приложения каталога для СВО
├── README.md           # Документация проекта
├── README_SVO.md       # Альтернативная документация проекта
├── package.json        # Файл зависимостей и скриптов
├── LICENSE             # Лицензия проекта
└── INSTALL.md          # Данный файл инструкции
```

## Возможности

- Интерактивный каталог по 10 макрогруппам профессий
- Фильтры по уровню образования, форме обучения и региону
- Информация о льготах для участников СВО
- Прямые ссылки на ФГОС и профессиональные стандарты
- Адаптивный дизайн для мобильных устройств
- Поиск по профессиям и специальностям
